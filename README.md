# Johannes Schulz - Portfolio

Personal portfolio website showcasing machine learning projects, skills, and professional experience.

## Features

- About Me section with professional background
- Interactive project showcases with live demos
- Skills and expertise overview
- Contact information

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- shadcn/ui components

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
